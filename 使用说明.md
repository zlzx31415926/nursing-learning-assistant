# 护理学习助手 · 使用说明

## 第一步：安装依赖

打开命令行（Win+R → 输入 cmd → 回车），输入：

```
pip install streamlit python-docx
```

## 第二步：设置 API Key

**方式A（推荐）：** 在 `api_key.txt` 文件里粘贴你的 DeepSeek API Key

**方式B：** 启动后在侧边栏输入

获取地址：https://platform.deepseek.com → 注册 → API Keys

## 第三步：启动

双击 `启动.bat`

浏览器会自动打开 http://localhost:8501

## 第四步：使用

1. 上传原始资料（.txt 或 .docx）
2. 点击「开始分析」→ AI 自动识别疾病并归组
3. 从左侧选择考点 → 自动生成六阶段学习环
4. 在六个选项卡中交互学习：
   - 阶段一：推理链逐个填写
   - 阶段二：基础填空 + 即时批改
   - 阶段三：主动预判陷阱 + 对比答案
   - 阶段四：临床病例决策
   - 阶段五：送命题测试 + 错题自动记录
   - 阶段六：错题画像 + 薄弱分析
5. 使用「深挖」按钮展开任意模块
6. 使用「粘合」按钮串联两个考点

## 文件结构

```
学习助手/
├── 学习助手.py         主程序
├── config.py           配置文件
├── 启动.bat            一键启动
├── api_key.txt         API Key（自行创建）
├── knowledge_base/     生成的学习材料
├── user_model.json     学习数据
└── error_log.json      错题记录
```
